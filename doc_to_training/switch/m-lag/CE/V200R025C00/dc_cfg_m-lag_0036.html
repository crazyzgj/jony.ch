Monitoring the M-LAG Operating Status

Context

During M-LAG operating status monitoring, you can check causes for fault locating if an M-LAG fault occurs.

Procedure

  1. Run the display m-lag troubleshooting { history | current }command to check causes for the M-LAG faults.

    This command can display the causes of a maximum of 100 recent faults at most.
